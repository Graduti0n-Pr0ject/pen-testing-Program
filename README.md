## Run
1. clone repo

    ```shell
    git clone https://github.com/Graduti0n-Pr0ject/pen-testing-Program.git
    cd pen-testing-Program
    ```
   
2. Install requirements

    ```shell
    pip install -r requirements.txt
    ```
   
3. go to directory `Scripts`

   ```shell
   cd Scripts
   ```
4. Run program

    ```shell
    python Index.py
    ```
